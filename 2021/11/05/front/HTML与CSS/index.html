      DIV简介

      什么是DIV

      DIV是层叠样式表中的定位技术，全称DIVision

      有时候把div称为图层，更多时候称为”块”

      DIV样式设置

      DIV设置大小、位置、背景

      div在没有设置宽度和高度时候

      宽度是上一级元素的宽度
      高度是div中内容的高度
      div1{
background-color: yellow;
width: 150px; #设置div宽度
height: 150px; #设置div高度
position: absolute; #绝对定位  没有该项后边top等不会生效
top: 150px; # 设置div起始位置
left: 150px; # 设置div起始位置
overflow: scroll; # 显示滚动条
}
      div块区域 overflow 属性设置

      visible 默认值 内容不会被修剪，超出部分呈现在边框外

      hidden 超出部分被隐藏

      scroll 不论是否超出 都显示滚动条

      auto 按需求显示滚动条以便查询其余内容

      div块轮廓（边框） outline
      div在没有设置宽度的情况下
      他的宽度是上级元素的宽度
      高度则是div中内容的高度
      作用：绘制元素周围的一条线，位于边框边缘的外围

      可以起到突出元素的作用

      none 无边框

      dashed 虚线边框

      dotted 点状边框

      solid 实线边框

      double 双线边框

      可以具体到上下左右边框

      border-left、border-right、border-top、border-bottom

      边框线是根据div块决定的 溢出部分失效

      盒子模型（改变内外间距）

      作用：在进行页面布局时，盒子模型非常重要

      margin：外边距

      margin：10px 10px 10px 10px 按照上右下左的顺序

      padding：内边距

      border：边框

      都可以分为 top bottom right left 四个方向

      box-sizing padding的距离产生：

      1、content-box：在宽度和高度之外绘制元素的内边距和边框

      （即额外延伸出去的距离）

      2、border-box：已设定的宽度和高度分别减去边框和内边距

      才能得到内容的宽度和高度

      （即div不变 将内容缩小平移）

      浮动float（重要）

      行级元素：行内元素和其他行元素在一条水平线上排列的

      （即不换行的）

      a标签 label img span等

      块级元素：在默认情况下，会独占一行

      div h标签 li table等

      定位机制：主要对于块级元素来说

      （标准）文档流、脱标流（float、position:absolute）

      float的四个属性：

      float：none 没有浮动

      float：left 左浮动

      float：right 右浮动

      float：inherit 继承父元素的浮动

      float的特性（重要理解）

      ①float的破坏性、崩塌性：

      父元素div只会被子元素中属于文档流的子元素撑起来，而不会被脱标流子元素撑起

      重要：将子元素设置为浮动后，父元素在没有指定高度的情况下，父元素的高度将不存在，如果想要使父元素具备重新具备高度，可以使用①给父元素手动添加高度，②清除浮动两种方法
      (重点)清除浮动方法：
      1、在浮动元素的最后添加clear：both；属性
      2、给父元素添加overflow：hidden；

      ②float的包裹特性

      当div产生了浮动后，div的宽度会自适应里面的内容，产生包裹特性

      定位

      position：relation相对、absolute绝对、static无定位、fixed固定

      绝对定位：相对自己最近的一个父级元素（容器）的定位 会产生浮动效果 脱离文档流 原位置会被其他div或者文字填充

      相对定位：相对自己本身原来的的位置 会保留之前停留的位置 原位置不会被其他div或者文字填充

      float 和 绝对定位的区别

      绝对定位 和 float 都会让元素脱离文档流

      但是绝对定位会将其他div或者文字元素覆盖

      而float不会覆盖 这就是区别
