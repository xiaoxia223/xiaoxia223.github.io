正则表达式

字符匹配
  • 一般字符 匹配自身 py python
  • . 匹配任意字符（换行符\n除外） py.hon python
  • \ 转义字符，使其后面一个字符改变原来的意思 python\.org python.org
  • [….] 字符集[a-c] [abc] p[xyz]t pxt pyt pzt
字符集匹配
  • \d 匹配 0-9的数字 a\dc a1c
  • \D 匹配所有非数字 a\Dc abc
  • \s 匹配任何空白字符 （空格 \r\n\t等） a\sc a c
  • \S 匹配非空白字符 a\Sc abc
  • \w 匹配包括下划线在内的任何字符 [A-za-z0-9_] a\wc abc
  • \W 匹配非字母字符 即匹配特殊字符 a\Wc ac
数量词匹配

  • * 匹配前一个字符0次或者多次 abc* ab; abccc

  • + 匹配前一个字符1次或者多次 abc+ abc;abccccc

  • ? 匹配前一个字符0次或者1次 abc? ab; abc

  • {} {m}匹配前一个字符m次， {m,n} 匹配前一个字符m-n次，若省略n 则匹配m至无限次 pyt{1,4}hon python;pytttthon

边界匹配

^ 匹配开头 $ 匹配结尾

逻辑或

| 或。匹配|左右表达式任意一个，从左到右匹配 abc|def abcdef

自动化测试工具 selenium库 自动化爬虫

为什么要用到selenium，有些网站数据是使用JS动态渲染过的，requests不能直接渠道数据

因此用到自动化工具selenium，

使用selenium模块调用浏览器渲染相关页面，然后使用正则或则xpath抽取响应的数据

selenium模块的使用前提是 webdriver驱动和浏览器版本相匹配

部分浏览器会自动更新，导致webdriver驱动不能使用 及时更换

pip install selenium 后需要下载个浏览器webdriver驱动并且配置环境变量才可以使用

selenium的简单实用 打开百度一下输入python并且点击访问

selenium元素定位方式

Selenium提供8中定位方式

id、name、class name、tag name、link text、partial link text、xpath、css selector

常用的就是id name 以及xpath

Selenium库浏览器交互操作技巧

可以操控浏览器的宽度高度、以及刷新、前进、回退等

selenium控制鼠标方法封装到了ActionChains类中

1、perform()：执行所有ActionsChains中存储的行为

2、context_click() 右击

3、double_click() 双击

4、drag_and_drop() 拖动

5、move_to_element() 鼠标悬停

selenium设置元素等待

WebDriver提供了两种类型的等待：显式等待和隐式等待

显式等待，就是明确的要等到某个元素的出现或者是某个元素的可以点击等条件，等不到，就一直等，除非在规定时间之内都没有找到，那就跳出Exception

隐式等待，当脚本执行到某个元素定位时，如果元素可以定位，则继续执行；如果元素定位不到，则它将以轮询的方式不断的判断元素是被定位到
