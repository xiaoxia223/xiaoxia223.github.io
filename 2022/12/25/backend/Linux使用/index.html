Linux知识

发表于2022-12-25|更新于2024-03-16|Linux
|字数总计:2.6k|阅读时长:8分钟

linux命令

  • 查看网卡：ifconfig
  • 关闭/打开网卡：sudo ifconfig 网卡名 down/up
  • 切换root用户 sudo su
  • 查看当前用户 whoami
  • 命令 date、ls(list) 展示当前目录：ls -a （a=all）
  • 短参数用一个- 长参数用两个–
  • 短参数可以合并 长参数不能合并
  • history 查找之前输入过的所有命令
  • ctrl + a 行首
  • ctrl + e 行尾
  • ctrl + d 清屏
  • ctrl + u 删除当前光标左侧所有字符
  • ctrl + w 往前删除一个单词
linux根目录下的直属目录

  • /bin目录 binary缩写 表示二进制文件 可执行程序

  • /etc 相当于and so on 一般放配置文件

  • /home 普通用户目录

  • /root 超级用户目录

  • /lib library的缩写 表示库

  • /media 媒体

  • rm、mv、cat、mkdir、cp、less、more、head、tail、touch、du用于统计目录大小

  • rm -r 可以删除目录 r是recursive的缩写 表示“递归的”

  • 硬链接、软链接

  • 硬链接相当 两个文件指向同一块内容 两个文件编号相同 删除一个另一个不受影响

  • 软链接相当于创建一个快捷方式 删除原文件后 快捷方式自然失效

用户权限、组

  • 需要超级管理员用户 sudo su

  • 创建新用户 adduser xiaoxia

  • 修改密码 passwd xiaoxia

  • 删除用户 deluser xiaoxia

组

  • 创建用户后 默认会设置一个组名和用户名相同的组

  • 创建组 addgroup friends

  • 将用户添加到组 usermod -g friends xiaoxia (mod=modify修改的意思)

  • 删除组 delgroup

  • 改变文件所有者 chown = change owner

  • 改变组文件群组 chgrp = change group

改变权限 chmod

  • d-rw-r–r– 第一个字符表示文件是目录还是其他 ，d表示目录，l表示链接，-表示普通文件

  • r read 只读、 w write 写权限 、 x execute 可执行

  • 前三个-rw表示所有者的权限 中间三个r–表示组用户的权限 最后三个r–表示其他用户的权限

  • linux对对应权限使用数字代替 r–4 w–2 x–1

    • — 0+0+0 = 0 r– 4+0+0=4 -w- 0+2+0=2
    • 640 表示 rw–w—-
    • 777表示所有权限 最宽权限 000 表示最低权限 没有任何权限
    • 使用方法 chmod 600 test.txt 即给test文件修改权限为 rw——-
命令使用手册

  • man命令 manual 使用手册的意思

  • man mkdir man cd 查找命令用法手册

  • apropos sound 查找跟sound有关的命令

  • whatis命令 是man命令的精简版 只显示概述命令

文件查找命令 locate和find

  • locate file 查找文件包含file关键字的所有文件和目录，有缺点 刚刚创建的文件或目录查不到

  • locate不会对整个硬盘查找. 而是从文件数据库中查找。

  • linux每天会更新一次文件数据库 刚创建的文件不会立马被加入文件数据库

  • 但是可以用updatedb命令强制系统立即更新文件数据库 只能root用户执行

  • find命令非常强大 不是查询数据库 而是遍历整个磁盘 所以不存在找不到刚刚创建的文件

  • find用法 find 何处 何物（必填） 做什么

VIM使用技巧

新建/打开文件： vi/vim 文件名
如果有文件则打开 没有文件则创建文件

打开文件并指定行数 ：vi 文件名 (+)行号 例如 vi .py +15
如果不添加行号 只剩+号 会直接定位到末尾

vi 三种模式

  1. 命令模式（使用vi打开文件后即进入命令模式，是vi的入口）

  2. 末行模式 （在命令模式下输入：进入末行模式）

  3. 编辑模式（在命令模式下输入i（insert插入）进入编辑模式）

  • 同样在末行模式或者编辑模式中按Esc进入命令模式

  • 编辑模式和末行模式不能直接切换 需要先切换到命令模式

移动光标等操作

  • hjkl 分别对应左下上右光标的移动

  • w（word）向后挪动一个单词

  • b （back）向前挪动一个单词

  • 0 表示回到该行行首

  • $ 表示回到该行行末

  • ^ 回到行首第一个不是空白字符的位置

行数移动
  • gg(go)文件顶部
  • G(go)文件尾部
  • 数字gg/G移动到指定行数
  • 末行模式 : 加数字 移动到指定行数
屏幕移动
  • ctrl+b（back）向上翻页
  • ctrl+f （forward）向下翻页
  • H（Head）屏幕顶部
  • M（Middle）屏幕中部
  • L（Low）屏幕底部
段落移动
  • 一段代码写在一起是一个段落
  • { 左括号代表向上寻找一段
  • } 有括号代表向下寻找一段
  • 还有就是% 可以在一对括号中来回切换
标记
  • 开发时某一块代码可能需要稍后处理
  • 可以使用标记
  • m（mark）+任意a-z或A-Z中字母 进行标记
  • 查看后可以使用 ‘+标记字母快速回到标记行
可视模式（选中文本）
  • v（visual视觉的） 可视模式 从光标位置开始正常选中文本
  • V 可视行模式 选中光标经过的完整行
  • ctrl+v 可视块模式 垂直方向选中文本
撤销和恢复撤销
  • u（undo） 撤销上一次的命令
  • ctrl+r（redo）恢复撤销命令
删除文本（删除命令都可以与移动命令连用）
  • x（cut） 删除光标所在字符 可以使用nx 删除指定多个字符
  • d（delate）删除移动命令对应的内容
  • dd 删除光标所在行，可以复制ndd多行
  • D 删除到行尾
  • dw 从光标删除到单词末尾
  • d0 从光标删除到行首
  • d} 从光标删除到字段结束
  • ndd 从光标位置向下连续删除n行
  • d代码行G 从光标删除到指定代码行
  • d’a 从光标位置删除到标记a之间所有的代码
复制粘贴（vi中提供一个被复制文本的缓冲区）
  • y（移动命令）（copy）复制
  • yy 复制一行 可以 nyy复制多行
  • p（paste）粘贴
  • 其中yy 和dd 都可以使用相同的命令
删除操作即剪切操作
替换
  • r（replace）替换当前字符 （在命令模式下）
  • R（replace）替换当前行光标后的字符 （替换模式）在光标处一直向后替换
缩排（缩进）与重复执行
  • 向右增加缩进
  • << 向左增加缩进
  • .重复上次命令
查找命令
  • /str 查找str，在查找到指定内容后 可以使用n/N（next）查找上一个/下一个出现的位置
单词快速匹配
  • 向后查找当前光标所在单词
查找并且替换

插入命令

  • O 在当前行上方插入一个空行并且进入编辑模式
  • o 在当前行下方插入一个空行并且进入编辑模式
  • a（append） 在当前光标下一个位置处进入编辑模式
  • A （append）在当前光标所在行尾处进入编辑模式
  • i（insert）在当前光标位置处进入编辑模式
  • I（insert）在当前光标所在行首进入编辑模式
编辑命令和数字联用

遇到重复的字符 比如连续输入10个*

利用可视块模式给多行代码增加注释
末行模式命令扩展
  • :e.（edit）会打开vi内置文件浏览器，浏览当前目录下的文件
  • :n（new）文件名 新建文件
  • :w（write）文件名 另存为 将当前编辑的内容插入到指定文件中 并且不会切换文件

注意：切换文件之前一定要保证当前文件已经被保存

分屏命令 可以同时编辑和查看多个文件（末行模式）
  • :sp（split）横向增加分屏
  • :vsp（vertical垂直split）纵向增加分屏
切换分屏窗口都是基于ctrl+w快捷键的 w（window）
  • w（window） 切换到下一个窗口
  • r（reserve）互换窗口
  • c（close）关闭当前窗口但不能是最后一个窗口
  • q（quit）退出当前窗口，如果是最后一个窗口 则退出vi
  • o（other） 关闭其他窗口
