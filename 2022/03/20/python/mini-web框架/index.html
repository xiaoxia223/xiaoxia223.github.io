闭包 （重要）

两个函数嵌套定义，内部函数用到外部函数的变量值，这个变量和内部函数组成的空间叫做闭包。应用是可以当做实参传递，并且可以同时传递数据和函数。 可以实现装饰器

简单的说就是 函数嵌套 外函数嵌套内函数，内函数往往会用到外函数的变量，

内函数和外函数的变量组成的空间 就是一个闭包

如果要在函数内部修改全局变量 可以直接在函数中加入 global x 然后再定义x的值

如果要修改闭包外边定义的变量，那么可以在函数中加入 nonlocal x 然后再定义x的值(当然python2不支持，py3之后才能用)

装饰器（重要）如果不会装饰器 就不要说会python

写代码要遵循开放封闭原则：不能对原函数直接进行修改

闭包可以实现装饰器，装饰器的作用是在原函数不做修改的情况下，给函数增加功能

在程序执行时候，只要遇到@符号 就已经开始装饰了 在函数调用之前就开始装饰了

每装饰一次 相当于创建一个闭包 闭包内调用原函数

元类（了解）

  • 什么是元类 元类其实就是type type 是一切的源头

  • 通过type创建出来类object，然后通过类创建出实例对象

  • 元类是创建类的东西

  • 类是创建实例对象的东西

  • 装饰器相当于可以给函数添加功能

  • 元类相当于给类添加功能

  • orm 对象关系映射

  • 将操作数据库的方法改为操作对象

WSGI协议是：当浏览器请求动态资源时，服务器通过一定的方式向web框架请求资源然后返回给浏览器
